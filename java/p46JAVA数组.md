# ==JAVA数组==

## 静态初始化数组

```java
//完整格式
数据类型[] 数组名 = new 数据类型[]{元素1,元素2,元素3……};
eg:int[] ages = new int[]{1,2,3};
eg:double[] scores = new double[]{1.2,2,3};
```

```java
//简化格式
数据类型[] 数组名 = {元素1,元素2,元素3……};
eg:int[] ages = {1,2,3};
```

注意：

+ ==”数据类型[] 数组名"也可以写成“数据类型 数组名[]”==
+ 什么类型的数组只能存放什么类型的数据

基本原理：

例如：int[] ages ={11,12,13};

ages存放数组第一个元素的地址
